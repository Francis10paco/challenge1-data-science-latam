Analisis de tiendas
El propósito de este proyecto es determinar en qué tienda debería vender el Sr. Juan sus productos,Para ello se analizaron diferentes factores: ingresos totales, categorías más y menos vendidas,calificaciones promedio de los clientes, productos destacados y el costo de envío promedio.
Además, se generaron visualizaciones que permiten comprender mejor los patrones encontrados.
